## Giriş

RestoCafe, modern restoranlar için geliştirilmiş kapsamlı bir yönetim sistemidir. Bu kılavuz, sistemin temel özelliklerini ve nasıl kullanılacağını açıklar.

## Kullanıcı Rolleri

### Admin
- Sistem yönetimi
- Kullanıcı yönetimi
- Raporlama ve analiz
- Tüm özelliklere erişim

### Garson
- Masa yönetimi
- Sipariş alma ve düzenleme
- Ödeme alma
- Rezervasyon yönetimi

### Mutfak
- Aktif siparişleri görüntüleme
- Sipariş durumu güncelleme
- Mutfak bildirimleri

### Kasiyer
- Ödeme işlemleri
- Günlük kasa raporu
- Fatura yönetimi

## Sistem Gereksinimleri

### Tarayıcı Gereksinimleri
- Google Chrome (son sürüm)
- Mozilla Firefox (son sürüm)
- Microsoft Edge (son sürüm)
- Safari (son sürüm)

### Donanım Gereksinimleri
- Minimum 2GB RAM
- Stabil internet bağlantısı
- Tablet veya masaüstü bilgisayar

## Temel Kullanım

### Sisteme Giriş
1. Web tarayıcınızda sistem URL'sine gidin
2. E-posta adresinizi girin
3. Şifrenizi girin
4. "Giriş Yap" butonuna tıklayın

### Şifre Değiştirme
1. Sağ üst köşedeki profil menüsüne tıklayın
2. "Şifre Değiştir" seçeneğini seçin
3. Mevcut şifrenizi girin
4. Yeni şifrenizi girin ve onaylayın
5. "Kaydet" butonuna tıklayın

### Dil Değiştirme
1. Sağ üst köşedeki dil seçeneğine tıklayın
2. İstediğiniz dili seçin (Türkçe/İngilizce)

## Rol Bazlı Kullanım Kılavuzları

### Admin Kılavuzu

#### Kullanıcı Yönetimi
1. Sol menüden "Kullanıcılar" seçeneğine tıklayın
2. Yeni kullanıcı eklemek için "+" butonuna tıklayın
3. Kullanıcı bilgilerini girin:
   - Ad Soyad
   - E-posta
   - Rol
   - Şifre
4. "Kaydet" butonuna tıklayın

#### Raporlama
1. Sol menüden "Raporlar" seçeneğine tıklayın
2. Rapor türünü seçin:
   - Satış Raporları
   - Müşteri Analizleri
   - Stok Raporları
   - Finansal Raporlar
3. Tarih aralığını belirleyin
4. "Rapor Oluştur" butonuna tıklayın
5. Raporu PDF veya Excel olarak indirin

### Garson Kılavuzu

#### Masa Yönetimi
1. Ana ekranda masa düzenini görüntüleyin
2. Masanın durumunu renk kodlarından takip edin:
   - Yeşil: Boş
   - Kırmızı: Dolu
   - Sarı: Rezerveli
3. Masaya tıklayarak detayları görüntüleyin

#### Sipariş Alma
1. İlgili masaya tıklayın
2. "Yeni Sipariş" butonuna tıklayın
3. Menüden ürünleri seçin:
   - Kategoriye göre filtreleme yapın
   - Arama kutusunu kullanın
4. Ürün miktarını belirleyin
5. Varsa özel notları ekleyin
6. "Siparişi Onayla" butonuna tıklayın

#### Ödeme Alma
1. İlgili masaya tıklayın
2. "Ödeme Al" butonuna tıklayın
3. Ödeme yöntemini seçin:
   - Nakit
   - Kredi Kartı
   - Yemek Kartı
4. Tutarı girin
5. Varsa bahşişi ekleyin
6. "Ödemeyi Tamamla" butonuna tıklayın

### Mutfak Kılavuzu

#### Sipariş Takibi
1. Ana ekranda aktif siparişleri görüntüleyin
2. Siparişleri öncelik sırasına göre takip edin
3. Sipariş detaylarını görüntüleyin:
   - Masa numarası
   - Sipariş içeriği
   - Özel notlar
   - Sipariş zamanı

#### Sipariş Durumu Güncelleme
1. İlgili siparişe tıklayın
2. Durumu güncelleyin:
   - Hazırlanıyor
   - Hazır
   - Servis Edildi
3. "Güncelle" butonuna tıklayın

### Kasiyer Kılavuzu

#### Günlük Kasa İşlemleri
1. "Kasa" ekranına gidin
2. Günlük ödemeleri görüntüleyin
3. Ödeme tipine göre filtreleme yapın
4. Kasa raporunu oluşturun:
   - Toplam satış
   - Ödeme tiplerine göre dağılım
   - İptal edilen işlemler
   - Bahşişler

#### Fatura Kesme
1. İlgili siparişi seçin
2. "Fatura Kes" butonuna tıklayın
3. Fatura tipini seçin:
   - Bireysel
   - Kurumsal
4. Fatura bilgilerini girin
5. "Faturayı Oluştur" butonuna tıklayın

## Sık Karşılaşılan Sorunlar

### Bağlantı Sorunları
1. İnternet bağlantınızı kontrol edin
2. Tarayıcı önbelleğini temizleyin
3. Sayfayı yenileyin
4. Sorun devam ederse sistem yöneticinize başvurun

### Yazıcı Sorunları
1. Yazıcı bağlantısını kontrol edin
2. Yazıcı kağıdını kontrol edin
3. Yazıcıyı yeniden başlatın
4. Sorun devam ederse teknik desteğe başvurun
